**Trinity is now Apache 2.0** 🎉

The Polyform Noncommercial license is gone. Trinity is true OSI-approved open source — free for any use, commercial included, with an explicit patent grant. Run your agent fleet in production, on any cloud or your own hardware, no commercial license required.

What changed: LICENSE is the verbatim Apache 2.0 text, contributions are now inbound = outbound under §5, and the `trinity-cli` (PyPI) and MCP server package metadata now match. Enterprise modules (SSO, user management, SIEM export) remain a separate commercial offering — the core platform is fully open.

Live now on GitHub: https://github.com/abilityai/trinity
